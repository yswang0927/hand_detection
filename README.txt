hand_detection
==============

DESCRIPTION:
------------

This web page will display the following information about hand movement:
		X index (side to side motion)
		Y index (up and down motion)
		Z index (in/out, or forward/back motion)
		
	It will display a numeric value as well a string stating either (left/right, up/down,
	forward/back).
	
	
	
SETUP:
------

Place index.html and leap.js in the same directory on your web server
